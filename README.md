
# Aprendizagem SQL 

Este repositório contém a implementação de uma série de exercícios avançados de Banco de Dados desenvolvidos na disciplina ministrada pelo Prof. Matheus Raffael Simon no curso de Ciência da Computação.

O objetivo do trabalho foi aplicar conceitos práticos de modelagem, consultas SQL com JOINs, criação de views, stored procedures, triggers, auditoria e definição de estrutura de banco de dados (DDL).
